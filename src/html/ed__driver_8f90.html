subroutine ed_driver

This sets up the drive for calling the ED code.
.

Here we initialiaze a few definition used in the work for for a cluster:
\( s_i \): Index for site \(-i\)
\(| F \rangle = \Pi_i |s_i\rangle \): Fock state configuration
\( | F \rangle = \{| F_n \rangle \}\): Fock space
\( H = H^T + H^C \): Hamiltonian
\( N \): Size of Fock space = Dimention of \(H\)

\( h_{ij}^T, h_{ij}^C\) : Hopping table, Coulomb table
Read \(\langle s_j|h_{ij}^T |s_i\rangle \), \(\langle s_j |h_{ij}^C |s_i\rangle \) and \(|F\rangle \) from io.f90 file
Build \( H \) by calling build_ham_i from ham.f90

Calls ED solver:
ed_solver = 0 : full ED using full_diag.f90
ed_solver = 1 : Lanczos from anczos.f90
ed_solver = 2 : Arpack from arpack.f90

\(E_i\); Eigenvalue with index- \(i\)
\(|V_i\rangle = \sum_n a_n^i |F_n\rangle \) : Eigenvector with index- \(i\)
